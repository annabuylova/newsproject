Проект позволяет проводить аналитику новостей и актуальность
их повестки для различных регионов 

В архиве лежат следующие файлы:
1. Work.py
	Основной код. Код для запуска streamlit 
	В консоли Pycharm после установки streamlit 
	(pip install streamlit) написать: 
	streamlit run NewsProject.py
	У вас должна прогрузиться страница с заголовком 
	"Статистика по новостям". Помимо описания основных шагов проекта
	там также есть интерактивные элементы. При выборе не шаблонной
	опции нужно дать странице перезагрузиться. Это должно произойти
	автоматически, но вы можете воспользоваться кнопкой Run в правом
	верхнем углу, если этого не произойдёт
	В нём часто встречаются команды read_csv, to_csv, а также driver =...
	В случае проблем, нужно будет изменить путь до искомого файла.
2. ПроектПитон.py
	Часть кода (без дополнений к streamlit). Фактически это черновик, 
	но не замена streamlit. Его удобно запускать через Jupiter Notebook. 
3. NewsProject.Rmd
	Код R, ищущий взаимозависимости между популярностью конкретных запросов
	В основном файле, открытом через streamlit, приведен код и результат для 
	конкретного df. В файле можно обновить результаты для новых df
2. Папка data. 
	Датафреймы в этой папке получены с помощью кода. Рекомендую 
	пользоваться готовыми df, чтобы не тратить часы на парсинг.
	В случае ошибок стоит проверить пути к этим файлам.
3. Убедитесь, что у вас установлено следующее:
	pip install selenium
	pip install pymorphy2
	pip install -U pymorphy2-dicts-ru
	pip install geopy
	pip install plotly
	pip install rpy2
	а также geckodriver.exe (firefox для работы с selenium). При необходимости
	стоит поменять пути в driver  
4. Технологии
	1. pandas (втч groupby)
	2. веб-скреппинг (selenium, robobrowser)
	3. API json
	4. визуализация  с доп.библиотеками
	5. матвозможности numpy
	6. streamlit
	7. регулярные выражения
	8. R
	9. tidyverse
	10. допбиблиотеки (pymorphy и др.)
